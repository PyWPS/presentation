Installing
----------

 - Checkout *pywps-4*
 - Checkout *pywps-4-demo*
 - Install dependencies (*requirements.txt*)
 - Install pywps-4
 - Run the demo server


Folder structure
----------------

.. code-block:: bash

   pywps-4
   pywps-4-demo
   +-- demo.py
   +-- LICENCE.txt
   +-- processes          <- working folder
   |   +-- area.py
   |   +-- bboxinout.py
   |   `-- ...
   +-- pywps.cfg          <- configurations
   +-- requirements.txt
   +-- server.py          <- run server
   +-- setup.py
   +-- static
   +-- templates
   `-- tests
